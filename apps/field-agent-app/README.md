# Field Agent App

Flutter app for low-connectivity field intake.

Current foundation includes:

- email/password and Google sign-in flow
- live or demo-mode auth gate
- multilingual UI support for English, Hindi, and Marathi
- light, dark, and system appearance
- field need capture with consent and vulnerability fields
- account and settings section
- live need status stream when Firebase is configured

Next upgrades after Firebase setup:

- GPS autofill
- attachment upload
- retry-aware offline outbox
- media and OCR workflows
